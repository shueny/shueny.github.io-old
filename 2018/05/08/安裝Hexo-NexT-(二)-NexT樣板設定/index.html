安裝Hexo-NexT-(二)-NexT樣板設定

以下設定檔皆為next中的「_config」

主題版型樣式選擇

在NexT中有四種版型：Muse、Mist、Pisces、Gemini，可以在 Scheme Settings中設定，選擇想要的版型後再把該版型前的 “#” 拿掉。

  1. Muse

    • 桌機版
    • 手機版

  2. Mist

    • 桌機版
    • 手機版

  3. Pisces

    • 桌機版
    • 手機版

  4. Gemini

    • 桌機版
    • 手機版

ReadMore 設定

在樣式Theme的_config中，找到 auto_excerpt，將原本enable的false改成true。

在網路上也找到其他做法，可以參考 Hexo 自动添加 Read More 标记

favicon 網頁圖示設定

可以到 next\source\images 把原本的檔案換掉。

favicon:
  small: /images/favicon-16x16.png
  medium: /images/favicon-32x32.png
  apple_touch_icon: /images/apple-touch-icon-180x180.png
  safari_pinned_tab: /images/s-logo.svg
