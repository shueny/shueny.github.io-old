Owl Carousel plugin 介紹

在說明這次的需求之前，先介紹一下這個plugin吧!

Owl Carousel，一個很好用的圖片輪播套件，可顯示自適應(RWD)版面，在不同尺寸顯示不同個數；還支援手指觸摸操作的效果，可以在手機直接用手勢滑動輪播圖片。

官網
Owl Carousel Version2

特點

1. 有 CDN 支援！不用再下載檔案及上傳到網路空間，可直接引用 CDN 的檔案外連。
2. RWD，隨著不同螢幕大小做縮放
3. 支援多種特殊輪播模式(例如由右到左、錨點操控、從螢幕中央開始輪播)
4. 可設定循環輪播
5. 可以設定不同尺寸的畫面，圖片一次輪播的張數
6. 可用滑鼠、手指來拖拉圖片
7. 可輪播 Youtube 影片
8. 可自訂各種參數、提供 API
9. 免費、可商業使用

如何使用?

下載並嵌入於HTML的head中

從CDN引入使用

[Owl Carousel2 cdn] (https://cdnjs.com/libraries/OwlCarousel2)
