CSS Grid 介紹

CSS Grid是第一個專門為解決佈局問題而產生的系統，它是一個新的二維網格(row & column)佈局系統，而Flexbox是一維的布局系統，兩者可以相輔相成，讓我們可以更快速的建造出有彈性的布局。

CSS Grid 使用

Grid Container

在外層容器設置 display: grid or display:inline-grid。

Grid Gap

在容器中，直的物件稱column、衡的物件稱row；物件之間的空隙稱作gap，因此有grid-column-gap及grid-row-gap，兩者皆含的稱grid-gap。

{% asset_img css-grid-2.jpg CSS Grid Container %}

Grid Lines

在columns之間的稱為columns lines；在rows之間的稱為rows lines。
意思：此物件在容器中，佔了哪裡到哪裡的位置。
