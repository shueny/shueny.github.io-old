Index 檔案在哪裡? 怎麼「index.liquid」檔案不在 layout 裡面?
index 是一個 template，不是一個 page，所以在 template 下有個 「index.liquid」。

index.liquid

檔案設定

首頁內容的呈現是從 Shopify 後台中 Theme Editor 設定的。

  • 路徑：Online Store → Themes → Current theme → Customize

  • 使用 Theme Editor 編輯器，選擇「Home Page」設定要在首頁呈現的區塊 (section)，自動新增 (Add section) 與刪除

  • 點下 「Add」並設好所需的設定，即可新增想要的 section 在首頁上

  • 最後在「setting_data.json」中最下面可以看到現在首頁呈現的 sections 有哪些
