shopify | 商品所屬哪些 collections? (product in which collections?)

發表於 2019-01-17 | 分類於 Shopify | | 閱讀次數:

問題

商品往往會屬於多個 collections 之下，要如何取得該商品所屬哪些 collections，以便做延伸應用?

解法

在 product-template (商品頁的樣版中，或是設定商品頁的檔案中) 使用
如此即可取出，此商品所屬在哪些 collections 之中。
參考資訊： product.collections

應用

有兩個活動(collections)，依照商品所屬在不同的活動(collections)之下，顯示不同的活動資訊在該商品頁中

作法

# Shopify # shopify syntax # shopify product # shopify collection
