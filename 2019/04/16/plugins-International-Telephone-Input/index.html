[Plugin]InternationalTelephone-Input 用於輸入和驗證國際電話號碼的 Plugin

Veröffentlicht am 2019-04-16 Bearbeitet am 2019-04-23 in Plugin

官網：International Telephone Input
Github：jackocnr/intl-tel-input

一個支援國際電話輸入的 jQuery Plugin

特性

  1. 在輸入國家代碼後，會自動顯示相對應的國家國旗
  2. 有電話號碼位數的控制
  3. 只可以輸入數字
  4. 國碼有下拉選單，選單可以用上下鍵控制
  5. 有初始化、設定好的選項
  6. 輸入完之後自動驗證
  7. 自動抓取 ip，顯示相對應的國家/國旗
  8. 輸入電話號碼，自動代上國碼
  9. 國家/地區同步
  10. 僅顯示某些國家 (例如：只顯示歐洲)
  11. 進入畫面時，顯示 value 中代入的號碼
  12. 顯示多個 input

欄位設定

  1. initialCountry：設定初始國家
  2. onlyCountries：設定只顯示的國家
