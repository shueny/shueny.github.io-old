[Vue] Error - TypeError: Cannot read property '__ob__' of undefined

نُشر في 2020-04-09 في Vue.js

在做上傳圖片的功能時，從動態訊息列表進到上傳畫面時，出現了 data functions should return an object。之後按下發布按鈕後，回到動態訊息列表時，也出現了 TypeError: Cannot read property ‘ob‘ of undefined 的錯誤訊息。

而我的 script 程式碼如下…

檢查了傳值也沒有錯、component 設定好像也沒錯?
google 查了一下才發現，原來是我沒有用到 data(){} ，但又將他留在程式碼中且沒有做 return，因此出現這兩個錯誤。

原因

data 在模擬數據時，data(){return{}} 這是需要 return 的。(如下程式碼)

解法

  1. 把 data(){} 刪掉
  2. 在 data 中補上 return{}，如: data(){return{}}
